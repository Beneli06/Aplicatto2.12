Apps/Frontend

Mover aquí los archivos del frontend: `index.html`, `index.tsx`, `App.tsx`, `components.tsx`, `views.tsx`, `services/*`, `types.ts`.

Ejemplo de estructura dentro de `apps/frontend/src`:
- src/
  - components/
  - pages/
  - services/
  - styles/
  - index.tsx
  - App.tsx

Mantén las llamadas a 3ros (GenAI) en el backend; el frontend solo debe llamar a `/api/v1`.
